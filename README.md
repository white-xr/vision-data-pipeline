# Vision Data Pipeline

通用视觉数据处理与训练工具仓库，面向数据清洗、数据筛选、数据增强、标注格式整理、数据集划分、YOLO 训练、模型评估和推理测试等流程。

## 目录约定

仓库只提交可复用代码、配置、文档和小型示例文件。数据集、采集原始数据、训练输出、模型权重、大图片/视频和压缩包不提交到 Git。

常见本地目录：

- `datasets/`：整理后的数据集
- `raw_data/`：原始数据
- `captures/`：相机采集结果
- `weights/`：模型权重
- `runs/`：训练与评估输出

这些目录已在 `.gitignore` 中忽略。

## 维护方向

- 数据清洗

- 数据增强
- 标注格式整理
- 数据集划分
